~Welcome.:).

Fill in the colour value boxes with R(ed), G(reen) and B(lue) values, or press Random Colour to generate a colour ( ..may hurt your head a bit:\ )
then press draw to generate a block (from the pixel size currently set to 50) or press loop to run along the grid
(the grid can be changed using the dimension boxes, which are currently set at 610 x 610 --
this will be the size of the .PNG file that you can export your image as by pressing the save image button
~hint: try setting the size to your phone or tablet screen dimensions to generate a :)background~ .!
~~hint 2: try making the red, green and blue values aprox ratios of each other. eg,: R:40, G:160 (40x4), B:80 (40x2):))

enjoy !
